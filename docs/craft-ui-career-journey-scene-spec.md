# 横版职业旅程场景规格

> 组件名称：`CareerJourneyScene`
>
> 定位：个人职业回顾页底部的互动装饰场景，用横版闯关游戏的语言重新讲述职业成长史。

## 1. 设计意图

场景介于叙事动画和轻互动之间，不做完整游戏系统。角色沿着时间轴跑过实习纪念节点，遇到节点后触发信息卡，再继续前进。跑到 `2026.06.05` 之后，以“时光倒流”方式循环。

整体使用手绘素描风，与 Journal 纸张质感一致：

- 所有形状以线条描边为主。
- 不使用大面积填色。
- 体积感使用点状笔触表达。
- 只有瞳孔、鼻子等极小功能细节可以使用实心点。

## 2. 画布与分层

- 固定高度约 `220px`。
- 全宽铺满页面底部。
- 背景完全透明。
- 地面位于高度约 `78%` 处。
- 世界宽度约 `4800` 单位，循环播放。

| 层级 | 内容 | 视差速度 |
| --- | --- | --- |
| 远景 | 云朵 | `0.26x` |
| 中远景 | 飞鸟 | `0.35x` |
| 中景 | 树木 | `0.70x` |
| 近景与地面 | 地面、刻度、小动物、Boss、花朵 | `1.0x` |
| 固定前景 | 奔跑角色 | 屏幕左侧约 `22%` |

## 3. 奔跑角色

首版使用拟人化线稿小熊。

外形：

- 圆形熊头、半圆耳朵和耳内小点。
- 小鼻子、眼睛和微笑。
- 简笔画身体与四肢。
- 只使用线稿，不使用大面积填色。

动作：

- 身体前倾约 `6deg`。
- 头部随步伐轻微上下弹跳。
- 双臂前后摆动。
- 双腿交替迈步，后脚踢起。
- 脚底落地时产生烟雾粒子。

后续使用 `GridFrameAnimation` 替换为手绘逐帧角色，不改场景调度层。

## 4. 时间轴

- 地面线下方绘制日期刻度，范围 `2025.10.10` 到 `2026.06.05`。
- 主刻度：起点、终点和关键月份使用较长竖线与日期标签。
- 副刻度：月度节点使用较短浅线。
- 左上角实时显示当前日期。

## 5. Boss 与信息卡

实习纪念节点：

| 日期 | 标题 | 说明 | 标记 |
| --- | --- | --- | --- |
| 2025.10.10 | 第一次坐到你们身边 | 实习开始 | 第 1 天 |
| 2025.12.01 | 普通日子慢慢变熟 | 一起吃饭、吐槽、赶进度 | 冬天 |
| 2026.03.15 | 把忙碌折进手帐 | 那些后来会想念的小事 | 春天 |
| 2026.06.05 | 这一页先写到这里 | 离开前的一封纪念信 | 第 239 天 |

Boss 外形：

- 圆形身体轮廓。
- 五根尖刺。
- 圆点眼睛与 V 形愤怒眉毛。
- 两条细腿。

触发流程：

```text
角色接近
  -> Boss 轻微抖动并略微放大
  -> 角色到达
  -> 8 条射线爆炸粒子向外扩散
  -> Boss 消失
  -> 信息卡从碰撞位置飞出并落到展示区域
  -> 信息卡停留约 3 秒后淡出
```

## 6. 场景装饰

树木：

- 每约 `490px` 世界宽度放置一组。
- 树干垂直生长，含 4 到 5 根侧向分支。
- 分支交替向左右上方生长，末端继续分叉。
- 每棵树高度、分支角度由种子决定，跨帧保持稳定。

云朵：

- 多段圆弧拼合。
- 只描边，无填充。
- 大小错落，低透明度慢速漂移。

飞鸟：

- 侧视线稿：椭圆身体、圆头、三角喙、翅膀弧线和尾羽。
- 翅膀使用正弦波扇动。
- 身体内部可加少量点状笔触。

地面小动物：

- 兔子：圆形身体、细长耳朵和圆尾巴，持续轻跳，起跳时轻微前倾。
- 青蛙：扁椭圆身体、圆眼和后腿，约 7 秒周期偶发跳跃。
- 青蛙起跳时纵向拉长，落地时横向扁压。
- 使用确定性种子错开节奏，不同步跳跃。

手帐纪念物：

- 沿时间轴散落便签、拍立得、票根、胶带、星光贴纸、信封、纸飞机、工牌、相机、咖啡杯和餐盘。
- 这些元素跟随世界坐标滚动，部分星光使用更慢视差。
- 所有位置、类型、尺寸与轻微旋转由固定种子生成，跨帧稳定。
- 纪念物只补足画面密度，不遮挡日期刻度、角色、节点信息卡和可点击种花反馈。

## 7. 点击种花

用户点击场景任意位置后种下一朵花：

1. 种子下落约 `0.15s`。
2. 茎干生长约 `0.55s`，中途萌出叶片。
3. 六片花瓣展开约 `0.37s`。

花朵落在世界坐标中，随后跟随场景滚动。花朵数量设置上限，超出后回收最旧实例。

## 8. 奔跑烟雾

- 每次脚底落地发射 2 个粒子。
- 粒子向后上方扩散。
- 带水平空气阻力和轻微重力。
- 使用线条圆圈，逐渐膨胀并淡出。
- 生命周期约 `0.55s`。

## 9. 未来区域

进入 `2026.06.05` 前的最后一段之后，逐渐出现中性墨色数字雨：

- 字符：`0`、`1`、`:`、`|`。
- 从右向左水平流动。
- 不使用绿色，不改变透明背景。
- 随进度逐渐增加密度。
- 后续可升级为“线条逐渐转成像素格、树木逐渐转成代码树”，避免 Matrix 风格与手帐语境割裂。

## 10. 时光倒流循环

周期结束后不做跳切，使用约 `1s` 的倒流过渡：

1. 世界视觉位置向右倒退，最大约 `90px`。
2. 日期数字随之倒退。
3. 半透明横向扫描线从下向上扫过。
4. 世界位置回到起点。
5. 扫描线快速消失，场景恢复正常。

禁止白色闪烁和背景色变化。

## 11. 视觉参数

| 用途 | 参数 |
| --- | --- |
| 全部线条 | `#2a2836` |
| 主线透明度 | `0.22` 到 `0.65` |
| 远景元素 | 使用更低透明度 |
| 背景 | 完全透明 |

Canvas 外层可叠加一层轻微 SVG 位移滤镜，模拟墨线不稳定感。滤镜参数缓慢变化，不能造成文字模糊或明显抖动。

## 12. 动画原则

| 原则 | 场景体现 |
| --- | --- |
| Squash & Stretch | 青蛙起跳和落地变形 |
| Anticipation | Boss 接近时抖动 |
| Staging | 地面、树木、云朵清晰分层 |
| Follow-through | 兔子起跳前倾，落地复位 |
| Overlapping Action | 动物节奏互相错开 |
| Ease In/Out | 种子下落和花朵生长 |
| Arcs | 爆炸粒子和跳跃轨迹 |
| Secondary Action | 墨线轻微抖动 |
| Timing | Boss 爆炸有显现和消失节奏 |
| Exaggeration | 奔跑摆臂和头部弹跳 |
| Solid Drawing | 近景粗、远景细 |
| Appeal | 大头角色和圆润动物 |

## 13. 交互

| 用户行为 | 场景响应 |
| --- | --- |
| 点击任意位置 | 种下种子并长出花朵 |
| 被动观看 | 自动滚动，角色奔跑并依次触发节点 |

场景不响应页面滚动、悬停和键盘。点击是唯一主动交互，留下的花朵是可见印记。

## 15. 性能预算

- 场景离开视口或页面隐藏时停止 `requestAnimationFrame`。
- 所有随机布局使用种子生成，禁止每帧随机重建。
- 烟雾粒子上限：`48`。
- 爆炸粒子上限：`40`。
- 花朵上限：`24`。
- 数字雨字符流上限：桌面 `36` 条。
- Canvas 使用设备像素比上限，避免高分屏过度绘制。
- 同屏只允许一个持续运行的特色动画区域。

## 16. 已知改进方向

1. 首版数学驱动小熊后续替换为精灵图。
2. 每个公司节点可以设计独立 Boss 主题。
3. 信息卡从碰撞位置飞出，增强层次感。
4. 增加动物种类和密度，但保持节奏错开。
5. 将未来区域升级为更统一的“代码树”表达。
6. 声效保持可选，默认关闭。
