# 239 Days Together

写给第一段实习里最好朋友们的纪念手帐网页。

## 本地运行

```bash
npm install
npm run dev
```

## 替换内容

- 寄语、时间线与媒体路径：`src/content/journey.ts`
- 本地照片和视频：`public/media`
- 项目约定与设计规范：`Agent.md`

## 第三方组件

本项目复制并使用了 [SuneBear/craft-paper](https://github.com/SuneBear/craft-paper)
中的 `paper-shape` 核心组件。许可文本见 `LICENSE.craft-paper`。
